# 📚 Book Recommendation System

## 🚀 Overview

This project is a **Content-Based Book Recommendation System** built using **Python and Machine Learning**. It suggests books to users based on similarity between different book features such as title, authors, categories, and descriptions.

Along with recommendations, the project also includes **data analysis and classification models** to better understand and predict book ratings.

---

## 🎯 Objectives

* Build a recommendation system based on book similarity
* Perform data analysis to extract meaningful insights
* Apply machine learning models for rating classification
* Demonstrate an end-to-end data science workflow

---

## 📂 Dataset

* 📎 Source: Kaggle
* 📁 File Used: `Books.csv`

```
https://www.kaggle.com/datasets/abdallahwagih/books-dataset/data
```

### 📊 Dataset Features

The dataset includes:

* 📖 Book Title
* ✍️ Authors
* 🏷️ Categories
* 📝 Description
* ⭐ Average Rating
* 👥 Ratings Count
* 📄 Number of Pages

---

## ⚙️ Features

### 🔹 Recommendation System

* Combines multiple book attributes into a single representation
* Finds similarity between books
* Recommends top similar books based on user input

### 🔹 Similarity Computation

* Converts data into numerical form
* Uses similarity measures to compare books

### 🔹 Exploratory Data Analysis (EDA)

* Distribution of ratings
* Insights from book-related data
* Visualization of trends


## 📊 Model Evaluation

* Evaluated using:

  * Accuracy
  * F1 Score
  * ROC-AUC

* Compared different models to determine best performance
